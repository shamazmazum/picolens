API

view(lens x)
Get the target of a lens, a getter or a fold.
set(lens val x)
Replace the target of a lens with val.
over(lens f x)
Modify the target of a lens applying f to it.
fold(f &rest gs)
Make a fold out of multiple getters gs and an associative binary operation f:
(view (fold #'* *car* *cdr*) '(2 . 3)) ; => 6
compose(&rest ls)

Compose lenses.

For example 

(view (compose (to #'cdr)(to #'car)) x)

is equal to 

(car (cdr x))
empty(fmap f x)
Neutral element in monoid of lenses under composition:
(equalp (view empty x) x)               ; => T
(equalp (over empty f x)(funcall f x)) ; => T
lens(get set)

Make a lens out of a setter of type a -> b -> c and a getter of type b -> a.

Example: 

(lens #'car (lambda (v cons)(cons v (cdr cons))))

is a lens equivalent of car.

to(get)
Make a getter out of a function get. An attempt to use this getter with over or set results in a runtime error.
gen-lenses(constructor &body lens-specs)
Generate lenses for a container (e.g. a structure, a class or even a cons cell). This macro is called with constructor which is the name of a constructor of the container followed by lens specification in the form (reader-name lens-name) for each slot (in the order determined by the constructor). Here is an example:
(serapeum:defconstructor node
  (value integer)
  (left  (or null node))
  (right (or null node)))

(gen-lenses node
  (node-value value)
  (node-left  left)
  (node-right right))
*car*
Lens equivalent of car
*cdr*
Lens equivalent of cdr
*caar*
Lens equivalent of caar
*cadr*
Lens equivalent of cadr
*cdar*
Lens equivalent of cdar
*cddr*
Lens equivalent of cddr
